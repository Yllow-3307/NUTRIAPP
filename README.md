# NUTRIAPP

Cette application est un tableau de bord de nutrition conçu pour les ectomorphes.

## Comment héberger ce site sur GitHub Pages

Pour que votre site soit accessible en ligne, suivez ces étapes :

1.  **Renommer le fichier principal** : C'est déjà fait, le fichier s'appelle maintenant `index.html`.
2.  **Activer GitHub Pages** :
    *   Allez sur la page de votre dépôt sur GitHub.
    *   Cliquez sur l'onglet **Settings** (Paramètres) en haut.
    *   Dans le menu de gauche, sous la section "Code and automation", cliquez sur **Pages**.
    *   Sous la section **Build and deployment** > **Branch**, assurez-vous que la branche sélectionnée est `main` (ou la branche par défaut) et que le dossier est `/ (root)`.
    *   Cliquez sur **Save**.
3.  **Accéder au site** : Après quelques minutes, GitHub affichera un message en haut de la page "Pages" avec l'URL de votre site (par exemple : `https://votre-pseudo.github.io/votre-depot/`).

## Fichiers techniques
*   `index.html` : Le code source de l'application (React).
*   `.nojekyll` : Empêche GitHub Pages d'ignorer les fichiers commençant par un underscore et optimise le chargement.
